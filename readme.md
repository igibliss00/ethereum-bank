## Installing

First, install Truffle globally.

>npm i -g truffle@nodeLTS

Install the relevant packages:

>git clone https://github.com/igibliss00/real_estate_erc721_dapp.git
>npm install

## Running the tests

To run all tests:

>truffle test

## Running the website
